# MYRA — направление «приложение как игра» + спутники (совещание 2026-07-22)

Рабочая записка по продуктовому направлению: сделать MYRA живее и игровее вокруг
спутника (питомца), с меньшим количеством текста и настоящими наградами.

## Голоса совещания

- **Гейм-дизайнер.** Движок уже есть (`useCompanion`): энергия за прослушивания,
  артефакты по XP/жанрам, дневной ритуал. Проблема — он спрятан в профиле. Вынести
  спутника в центр опыта: реагирует на музыку, растёт на глазах, награды на пути.
- **Продукт/UX.** «Много слов» = экраны объясняют текстом то, что должно
  показываться. Замена: прогресс-кольцо вместо абзаца, эмоция питомца вместо
  описания, один заголовок вместо трёх. Ритм «одно действие на экран».
- **Ретеншн.** Возвращают: видимый прогресс, streak/дневной ритуал (есть!),
  «завтра откроется артефакт». Надо сделать заметным (приветствие-возвращение,
  бейдж на иконке).
- **Бренд + честность.** Награды должны быть настоящими (оформление/плейлист/бейдж),
  а не пустые фантики — принцип честности проекта. Держим премиальную аврору,
  не «детский мультик».

**Спор:** питомец везде vs. не мешать музыке. **Компромисс:** тёплый спутник на
входе и в «доме», ненавязчив во время прослушивания.

## Решение (переиспользуем готовый `useCompanion`, не строим заново)

### Этап 1 — вход и первое касание
- [x] Экран выбора спутника (трио) — сделано.
- [x] **A.** Минималистичная настройка без ручного выбора жанров: экран «спутник
  настраивается под тебя»; вкус берётся из первых прослушиваний (`smart.ts`), без анкет.
- [x] **B.** Логин в стиле Яндекс-ID: спутник в скруглённом «аватаре аккаунта»,
  аврора-стекло, заголовок под режим, скользящий переключатель.

### Этап 2 — «дом» питомца
- [x] **C.** Приветствие при заходе: «Пока тебя не было, {спутник} собрала…» —
  настоящие рекомендации из `smart.ts`, кольцо роста + маркер следующей награды
  без текстовой шкалы. Показ раз в день, только вернувшимся (есть история).

### Этап 3 — награды по-настоящему
- [x] **Оформление плеера.** Надетый (показанный) артефакт реально оформляет
  плеер: свечение в его цвете + герб рядом со «Сейчас играет». В коллекции —
  честная подпись «Надеть на плеер» / «Надето · оформляет плеер».
- [x] **Эксклюзивный плейлист** за веху спутника (2 уровень): на Полке, играет
  реальные треки в открытых жанрах; честный lock/unlock с прогрессом.
- [x] **Бейджи-награды** в профиле: «Знаток» (3 уровень) и «Хранитель артефактов»
  (все артефакты собраны) — появляются только когда реально заработаны.

### Сквозные
- [x] Жанры переведены на русский во всём демо-каталоге (согласованно).
- [ ] Вычистка лишнего текста экран за экраном (меньше слов).

## Принцип
Ничего симулированного как реального; награды осмысленные; премиальная айдентика
(аврора/стекло/Sora), питомец — эмоциональный стержень, а не кликер.
